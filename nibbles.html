What to Include in Plan
NIBBLES approach:
  • Neurological Level: include AIS
  • Immobility: any orthoses? place nursing communication
  • Bladder: appropriate UMN vs LMN orders
  • Bowel: appropriate UMN vs LMN orders
  • Blood Pressure: consider orthostatic hypotension
  • Bone: vitamin D level, supplementation prn
  • Lungs: include orders for nebulizers, ventilator settings, chest perccusion, post drainage and insufflator use if any needed
  • Embolism: DVT Prophylaxis
  • Skin: wound care orders, turning schedule, also Lac-hydrin if dry skin
  • Pain: neuropathic? nociceptive? visceral?
  • MSK: include any pertinent MSK concerns
